### 1. What is the difference between null and undefined?

#### undefined

A variable that has been declared but not assigned a value. Type of undefined is "undefined".

#### null

Represents the absence of value which is assigned by the developer. Type of null is "object".

### 2. What is the use of map() in JavaScript? How is it different from forEach()?

#### map()

Creates a new array by transforming each element.
Returns a new array.

#### forEach()

Executes a function on each element.
Does NOT return a new array.

### 3. What is the difference between == and ===?

#### ==

Compares values after type conversion.

#### ===

Compares both value and data type.

### 5. Explain the concept of Scope in JavaScript

Scope determines where variables can be accessed.

#### Global Scope

Declared outside any function. Accessible everywhere.

#### Function Scope

Declared inside a function. Accessible only inside that function.

#### Block Scope

Declared inside {} using let or const. Accessible only inside that block.
